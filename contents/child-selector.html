子セレクタ

指定した親子関係を持つ要素に対してスタイルを変更することが出来ます。

child-selector.css を使用しましょう。

td#apple-ja li.odd {
    color: red;
}

1. td#apple-ja li.odd を指定して apple-ja id を持つ td の中にあり odd クラスを持つ li 要素の文字色を red にしてみましょう。
