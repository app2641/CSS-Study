表示と配置

指定要素の表示や配置についてスタイルを適用することが出来ます。

display-and-position.css を使用しましょう。

div#overflow {
    overflow: hidden;
}

div#overflow {
    overflow: scroll;
}

overflow はボックスに入りきらずはみ出た部分の表示方法を指定します。

1. div#overflow からはみ出た部分を非表示にしてみましょう。

2. div#overflow からはみ出た部分をスクロールして見られるようにしてみましょう。

ゴーシュは町の活動写真館でセロを弾く係りでした。けれどもあんまり上手でないという評判でした。上手でないどころではなく実は仲間の楽手のなかではいちばん下手でしたから、いつでも楽長にいじめられるのでした。

ひるすぎみんなは楽屋に円くならんで今度の町の音楽会へ出す第六交響曲こうきょうきょくの練習をしていました。

トランペットは一生けん命歌っています。

div#display {
    display: none;
}

3. div#display の要素を非表示にしてみましょう。

div#display

p#relative {
    position: relative;
    top: 20px;
    left: 20px;
}

position: relative は親要素からの相対位置でポジションを指定するという宣言です。

4. p#relative 要素をボックスの上から 20px 左から 20px の位置に配置してみましょう。

relativeポジションテキスト

p#absolute {
    position: absolute;
    top: 20px;
    left: 20px;
}

position: absolute はページ全体からの絶対位置でポジションを指定するという宣言です。

親要素に position: relative が指定されていた場合はその要素からの絶対位置になります。

5. p#absolute 要素をページの上から 20px 左から 20px の位置に配置してみましょう。

absoluteポジションテキスト

p#fixed {
    position: fixed;
    top: 40px;
    left: 40px;
}

position: fixed は absolute 同様ページ全体からの絶対位置でポジションを指定するという宣言です。

absolute と違うのはスクロールしても位置が固定されているというところです。
