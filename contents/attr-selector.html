属性セレクタ

指定属性の要素に対してスタイルを変更することが出来ます。

attr-selector.css を使用しましょう。

li[class] {
    color: red;
}

li[class="odd"] {
    color: blue;
}

1. li[class] を指定して class 属性を持つ要素の文字色を red にしてみましょう。

2. li[class="odd"] を指定して class 属性の値が odd の文字色を blue にしてみましょう。
