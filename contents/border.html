ボーダー

ボーダーに関わるスタイルを変更することが出来ます。

border.css を使用しましょう。

p#solid {
    border-style: solid;
}

p#double {
    border-style: double;
}

p#solid-and-double {
    border-style: solid double;
}

boder-style はボーダーの種類を指定します。

1. p#solid 要素に一本線を引いてみましょう。

2. p#double 要素に二重線を引いてみましょう。

3. p#solid-and-double 要素の上下に一本線、左右に二重線を引いてみましょう。

doubleボーダーテキスト

複合ボーダーテキスト

p#red-border {
    border-style: solid;
    border-color: red;
}

4. p#red-border の要素に赤色の一本線を引いてみましょう。

    border-style: solid;
    border-width: 1px;

p#em-border {

    border-width: 1em;

5. p#pixel-border 要素に太さ1pxの一本線を引いてみましょう。

6. p#em-border 要素に太さ1emの一本線を引いてみましょう。
