カーソル

カーソルのスタイルを変更することが出来ます。

cursor.css を使用しましょう。

div#pointer {
    cursor: pointer;
}

1. div#pointer 要素上でポインタカーソルに変更してみましょう。

div#pointer

div#move {
    cursor: move;
}

2. div#move の要素上で移動カーソルに変更してみましょう。

div#move

div#wait {
    cursor: wait;
}

3. div#wait の要素上で待機中カーソルに変更してみましょう。

div#wait

カーソルは他にも様々なものがあります。ぐぐって調べましょう。
