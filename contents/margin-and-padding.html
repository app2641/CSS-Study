マージンとパディング

指定要素の余白についてスタイルを適用することが出来ます。

margin-and-padding.css を使用しましょう。

div#margin {
    margin-top: 50px;
}

div#margin {
    margin-right: 50px;
}

div#margin {
    margin: 40px;
}

div#margin {
    margin: 20px 10px;
}

div#margin {
    margin: 5px 10px 0px 30px;
}

マージンは要素の外側にある余白に対してスタイルを適用します。

1. div#margin の上部の余白を 50px にしてみましょう。

2. div#margin の右部の余白を 50px にしてみましょう。

3. div#margin の上下左右の余白を 40px にしてみましょう。

4. div#margin の上下の余白を 20px に、左右の余白を 10px にしてみましょう。

5. div#margin の上 5px 右 10px 下 0px 左 30px の余白にしてみましょう。

div#margin

div#padding {
    padding-top: 50px;
}

div#padding {
    padding-left: 50px;
}

div#padding {
    padding: 40px;
}

div#padding {
    padding: 20px 10px;
}

div#padding {
    padding: 5px 10px 0px 30px;
}

パディングは要素の内側にある余白に対してスタイルを適用します。

6. div#padding の上部の余白を 50px にしてみましょう。

7. div#padding の左部の余白を 50px にしてみましょう。

8. div#padding の上下左右の余白を 40px にしてみましょう。

9. div#padding の上下の余白を 20px に、左右の余白を 10px にしてみましょう。

10. div#padding の上 5px 右 10px 下 0px 左 30px の余白にしてみましょう。

div#padding
