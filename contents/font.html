CSS study

フォント

指定要素にのフォントを変更することが出来ます。

font.css を使用しましょう。

p#font-bold {
    font-weight: bold;
}

p#font-italic {
    font-style: italic;
}

1. p#font-bold を太字にしてみましょう。

2. p#font-italic をイタリックにしてみましょう。

ボールド

イタリック

p#font-size-percent {
    font-size: 200%;
}

p#font-size-em {
    font-size: 2em;
}

p.line-height {
    line-height: 400%;
}

3. p#font-size-percent を 200% のサイズにしてみましょう。

4. p#font-size-em を 2.0em のサイズにしてみましょう。

5. p.line-height の行の高さを 400% にしてみましょう。

p#font-serif {
    font-family: serif;
}

p# font-impact {
    font-family: impact;
}

6. p#font-serif に明朝系のフォントを指定してみましょう。

7. p#font-impact に impact のフォントを指定してみましょう。

応用: WebFont という外部からフォントを読み込む機能もある。Google Fonts など。
