背景

背景に関わるスタイルについて説明します。

background.css を使用しましょう。

div#background-color {
    background-color: grey;
}

1. background-color で div 要素の背景色を grey にしてみましょう。

div#background-color

div#background-image {
    background-image: url('../../images/chrome.png');
    background-repeat: no-repeat;
    background-position: 20px 5%;
}

2. background-image で div 要素の背景に画像を配置してみましょう。

3. background-repeat で背景画像を単体で表示してみましょう。

4. background-position で背景画像の開始位置を変更してみましょう。

div#background-image

div#background-stripe {
    background-image: url('../../images/stripe.png');
}

5. background-image を応用して背景に柄を表示してみましょう。

div#background-stripe
