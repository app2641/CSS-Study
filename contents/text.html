指定要素のテキストにスタイルを適用することが出来ます。

text.css を使用しましょう。

p#align-right {
    text-align: right;
}

p#align-center {
    text-align: center;
}

1. p#align-right のテキストを右揃えにしてみましょう。

2. p#align-center のテキストを中央揃えにしてみましょう。

4. p#decoration-through のテキストに打ち消し線を引きましょう。
